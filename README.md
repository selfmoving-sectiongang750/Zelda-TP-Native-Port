## 🛠️ Troubleshooting

If the game does not start, confirm that you have the latest drivers for your graphics card. You can find these updates on the official website for your graphics card manufacturer. 

If you see a missing file error, verify that you extracted all files from the zip folder. Sometimes, security software blocks the executable file. If this happens, add the game folder to your security software's exclusion list.

If the game crashes upon launch, delete the settings.ini file inside the game folder. The software will create a fresh file with default settings the next time you open the game.

The software runs in a window by default. Press Alt and Enter at the same time to switch to full-screen mode. Press these keys again to return to windowed mode.
